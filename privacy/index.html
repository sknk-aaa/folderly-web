Frequently Asked Questions

The folder icon didn't change after I applied it

Windows Explorer caches folder icons. Folderly refreshes the target window automatically after applying. If the old icon still shows, close and reopen that Explorer window, or sign out and back in. You do not need to restart your PC.

Do I need to convert my image to .ico first?

No. Just drop a PNG or JPG (or pick one from the drop area) and Folderly creates the folder icon for you — no .ico conversion needed.

Which image formats are supported?

PNG, JPG/JPEG, BMP, and WebP.

Can I undo a change and restore the original folder?

Yes. Open Folderly, find the folder under History, and click Revert. The folder returns to its normal appearance.

I moved or renamed a customized folder and can't revert it

Because the folder's path changed, Folderly can no longer restore that exact folder automatically. Open Folderly, select the entry in History, and remove the history entry. If a leftover _folderly folder or desktop.ini remains inside the moved folder, you can delete them manually.

Is Folderly free?

Folderly is a paid app with a free trial. After the trial, you can buy it with a single one-time purchase — there is no subscription.

適用してもフォルダのアイコンが変わりません

Windows のエクスプローラーはフォルダアイコンをキャッシュします。Folderly は適用後に対象ウィンドウを自動で更新しますが、古いアイコンのままの場合は、そのエクスプローラーウィンドウを閉じて開き直すか、一度サインアウトして再ログインしてください。PC の再起動は不要です。

画像を .ico に変換する必要はありますか？

不要です。PNG や JPG をドロップエリアに入れる（または選ぶ）だけで、Folderly がフォルダアイコンを自動生成します。.ico への変換はいりません。

対応している画像形式は？

PNG、JPG/JPEG、BMP、WebP に対応しています。

元のフォルダに戻せますか？

はい。Folderly を開き、履歴から対象フォルダを選んで「元に戻す」を押すと、通常の見た目に戻ります。

カスタマイズ済みのフォルダを移動・改名したら解除できません

フォルダのパスが変わったため、Folderly はそのフォルダの実体を自動では復元できません。Folderly の履歴で該当エントリを選び、履歴を削除してください。移動先のフォルダ内に _folderly フォルダや desktop.ini が残っている場合は、手動で削除できます。

Folderly は無料ですか？

Folderly は無料試用版付きの有料アプリです。試用期間後は、買い切り（サブスクなし）で購入できます。

Privacy Policy

Data Collection

Folderly does not collect, transmit, sell, or share personal data.

Local Data

Folderly stores application settings, history, selected image copies, and generated icon files locally on your device. This data is used only to provide app features such as applying folder icons, restoring previews, showing history, and reverting customized folders.

Typical local data may include:

  • Folder paths customized by the user
  • Source image paths selected by the user
  • Local copies of selected images for preview restoration
  • Generated icon files
  • App settings such as language, tag names, and display options

Images

Images selected by the user are processed locally on the device. Folderly does not upload selected images to any server.

Analytics and Tracking

Folderly does not include analytics, advertising trackers, or telemetry.

Microsoft Store

If Folderly is installed through Microsoft Store, Microsoft may process purchase, licensing, crash reporting, and Store-related data according to Microsoft's own policies. Folderly does not receive this Store account data from Microsoft.

プライバシーポリシー

データ収集

Folderly は、個人データを収集、送信、販売、共有しません。

ローカルに保存されるデータ

Folderly は、アプリ設定、履歴、選択した画像のコピー、生成したアイコンファイルをユーザーのデバイス上に保存します。これらのデータは、フォルダアイコンの適用、プレビューの復元、履歴表示、元に戻す機能など、アプリ機能の提供にのみ使用されます。

保存される可能性があるデータ:

  • ユーザーがカスタマイズしたフォルダのパス
  • ユーザーが選択した画像ファイルのパス
  • プレビュー復元用に保存される選択画像のローカルコピー
  • 生成されたアイコンファイル
  • 言語、タグ名、表示オプションなどのアプリ設定

画像

ユーザーが選択した画像は、デバイス上でローカルに処理されます。Folderly は選択画像をサーバーにアップロードしません。

分析とトラッキング

Folderly には、分析、広告トラッカー、テレメトリは含まれていません。

Microsoft Store

Folderly を Microsoft Store からインストールした場合、購入、ライセンス、クラッシュレポート、Store 関連データは Microsoft のポリシーに基づいて処理される場合があります。Folderly は、これらの Store アカウント情報を Microsoft から受け取りません。
