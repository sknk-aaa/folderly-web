Windows 11でフォルダのアイコンを
画像に変更する方法

Windowsでエクスプローラーを開くと、似たような黄色いフォルダがずらり。「写真」「請求書」「去年の資料」——名前は違うのに見た目は全部同じで、目当てのフォルダを探すのにいちいち時間がかかる。そんなストレス、ありませんか。

実は Windows でも、フォルダのアイコンは好きな画像に変えられます。この記事では、まず無料でできる標準のやり方から、写真をもっと手軽に使う方法、そして「うまく変えられない」ときの対処まで、順番に・つまずかないように解説します。

はじめにそもそもフォルダのアイコンは変えられる？

結論から言うと、変えられます。しかも特別な知識はいりません。Windows には標準でアイコンを変える機能があり、追加のアプリを使えば「手持ちの写真をそのまま」フォルダの顔にすることもできます。

下の画像が、実際に変えるとどうなるかのイメージです。左が変更前、右が画像アイコンにした後。ひと目で中身が想像できて、探す時間がぐっと減ります。

やり方は大きく2通り。まずはお金のかからない標準の方法から見ていきましょう。

方法 1 ・ 無料Windows標準でフォルダアイコンを変更する

追加アプリなしでも、Windows の機能だけでフォルダのアイコンは変更できます。手順はシンプルです。

  1. 変更したいフォルダを右クリックして、「プロパティ」を開きます。
  2. 「カスタマイズ」タブを開き、下のほうにある「アイコンの変更」をクリックします。
  3. 表示された一覧から選ぶか、手持ちの .ico ファイルを指定して「OK」。これで完了です。

知っておきたい標準機能の「クセ」

手軽に見えますが、「手持ちの写真を使いたい」となると、いくつか引っかかる点があります。ここは正直にお伝えします。

  • 使えるのは .ico という専用形式だけ。スマホやカメラの写真（JPG / PNG）はそのままでは選べません。
  • 写真を使うには、いったん .ico に変換するツールを探して使う手間が必要です。
  • 指定した .ico ファイルを後で移動・削除すると、アイコンが元に戻ってしまいます（場所を覚えているだけだから）。
  • 表示の位置や大きさは微調整できません。

「ちょっと試してみるか」のつもりが、変換ツールを探すあたりで面倒になって挫折——というのは、実はよくある話です。

写真をそのまま、ドラッグするだけでアイコンにできたら。
そう思ったことがあるなら、次の方法が近道です。

方法 2 ・ かんたん写真をそのままアイコンにする

「変換は面倒。手持ちの写真をそのまま使いたい」——そんな人のために、私たちは Folderly という小さなアプリを作りました。.ico への変換は不要。画像をドラッグするだけで、フォルダのアイコンになります。

  1. フォルダを右クリックして、「Customize with Folderly」を選びます。
  2. 好きな画像（PNG / JPG）をドラッグ＆ドロップ。写真でもイラストでもロゴでもOK。
  3. プレビューを見ながら大きさや位置を調整して、「適用」を押すだけ。

標準機能で面倒だった点が、そのまま解決します。色タグやタグ名でフォルダを分類できるので、用途ごとに色分けして「探さずに見つかる」状態にもできます。気が変わっても、履歴からいつでも元に戻せるので安心して試せます。

困ったときは変更できない・元に戻る時の対処

「設定したのに反映されない」「いつの間にか元に戻っている」——よくあるつまずきと対処法です。

  • アイコンが元に戻ってしまう：標準の .ico 方式は、指定したファイルを移動・削除すると戻ります。元の場所に置いたままにするか、ファイルを内部で管理してくれる Folderly のような方法を使うと安心です。
  • 変更が反映されない・遅い：Windows はアイコンをキャッシュするため、すぐ反映されないことがあります。対象フォルダやエクスプローラーのウィンドウを開き直すと更新されます（Folderly は適用後に自動で更新します）。
  • そもそも写真が選べない：標準機能は .ico しか受け付けません。写真をそのまま使いたいときは Folderly を使ってください。
豆知識： 適用したアイコンは desktop.ini という仕組みでフォルダに保存されます。Folderly は元の状態を記録しているので、気が変わってもいつでも元のフォルダに戻せます。
