Windows 11でフォルダのアイコンを画像に変更する方法

2026-06-01 ・ Windows 10 / 11 対応

フォルダが全部同じ黄色で、目的のフォルダを探すのに地味に時間がかかる——そんな経験はありませんか。実は Windows でも、フォルダのアイコンは好きな画像に変えられます。

この記事では、(1) Windows 標準のやり方と、(2) 写真をそのまま使える専用アプリ「Folderly」での方法を、初心者の方にもわかるように解説します。最後に「アイコンが変更できない／元に戻ってしまう」ときの対処法もまとめました。

方法1：Windows標準でフォルダアイコンを変更する

追加アプリなしでも、Windows の機能だけでフォルダのアイコンは変更できます。手順は次の通りです。

  1. 変更したいフォルダを右クリック →「プロパティ」を開く
  2. 「カスタマイズ」タブ →「アイコンの変更」をクリック
  3. 用意されたアイコン、または手持ちの .ico ファイルを選んで「OK」

標準のやり方の弱点

手軽そうに見えますが、写真をアイコンにしたい場合はいくつか壁があります。

  • 使えるのは .ico 形式だけ。手持ちの写真（JPG / PNG）はそのままでは使えず、.ico へ変換が必要。
  • 画像を .ico に変換するツールを別途探す手間がかかる。
  • アイコンに使った .ico ファイルを移動・削除すると元に戻る（ファイルのパスに依存するため）。
  • 表示位置や拡大率の微調整ができない。

「ちょっとやってみるか」のハードルが、地味に高いんです。

方法2：写真をそのままアイコンにする（Folderly）

Folderly を使うと、.ico 変換なしで、画像をドラッグするだけでフォルダアイコンにできます。写真でもイラストでもロゴでもOKです。

  1. フォルダを右クリック →「Customize with Folderly」を選ぶ
  2. 好きな画像（PNG / JPG）をドラッグ＆ドロップ
  3. プレビューを見ながら拡大率や位置を調整し、「適用」を押すだけ

Folderly のポイント：

  • .ico 変換は不要。写真をそのまま使える。
  • 色タグやタグ名でフォルダを分類でき、用途で色分けして探しやすい。
  • いつでも元に戻せる（変更履歴から個別・一括で復元）。

フォルダアイコンが変更できない・元に戻る時は？

うまくいかない場合は、次の点を確認してください。

  • アイコンが元に戻ってしまう：標準の .ico 方式は、指定した .ico ファイルを移動・削除すると戻ります。元の場所に置いたままにするか、Folderly のようにファイルを管理してくれる方式を使うと安心です。
  • 変更が反映されない・遅い：Windows はアイコンをキャッシュするため、すぐに反映されないことがあります。対象フォルダやエクスプローラーのウィンドウを開き直すと更新されます（Folderly は適用後に自動でウィンドウを更新します）。
  • そもそも画像が使えない：標準機能は .ico しか受け付けません。写真をそのまま使いたい場合は Folderly を使ってください。
ヒント： 適用したアイコンは desktop.ini という仕組みでフォルダに保存されます。Folderly は元の状態を記録しているので、気が変わってもいつでも元のフォルダに戻せます。

まとめ

Windows 標準でもフォルダのアイコンは変更できますが、写真を使うには .ico 変換が必要で手間がかかります。「手持ちの画像をそのまま使いたい」「位置を調整したい」「あとで戻したい」なら、専用アプリの Folderly が圧倒的に簡単です。
