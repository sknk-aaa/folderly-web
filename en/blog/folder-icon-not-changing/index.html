Folder icon won't change
or keeps reverting?

You changed a folder's icon, but nothing happens. Or you set it perfectly — then a restart brings the plain yellow folder right back. If Windows icons keep fighting you, you're not alone: this trips up a lot of people.

The good news: the causes fall into a handful of predictable patterns, and once you know which one you're hitting, the fix is easy. Below, we go symptom by symptom — why it happens and how to fix it.

START HEREFirst: the four common causes

When changing a folder icon doesn't work, it's almost always one of these four. Spot which one matches your symptom, and you'll get to the fix faster.

  • ① The display cache — your setting is correct, but the old look is stuck on screen.
  • ② The file's location — you moved or deleted the file you used, so it reverts.
  • ③ The image format — you can't pick a photo because the built-in tool only takes .ico.
  • ④ The folder type or permissions — "Change Icon" doesn't show up at all.

If you haven't actually walked through the steps to change an icon yet, start with How to change a folder icon to an image on Windows 11 — this guide will make more sense afterward.

SYMPTOM 1The change doesn't show up

You clicked OK, but the icon looks the same. This is almost always the icon cache. To make things load fast, Windows caches icons, and sometimes the old one stays on screen. Your setting actually worked — you just need to refresh the display.

  1. Open the parent folder again, or press F5 in File Explorer to refresh.
  2. Still stuck? Open Task Manager (right-click the taskbar → Task Manager), select Windows Explorer, and click Restart.
  3. If it's still not showing, restart your PC. That clears it up most of the time.
Advanced: clear the icon cache manually.
End Windows Explorer in Task Manager, then in Command Prompt run del /a %localappdata%\IconCache.db and del /a %localappdata%\Microsoft\Windows\Explorer\iconcache*, and restart Explorer. Only if you're comfortable with the command line.

SYMPTOM 2It keeps reverting back

"I changed it, but a restart undid it." "It was back to plain yellow days later." This happens because the built-in method only remembers where the file is — it doesn't keep a copy. Any of these breaks the reference and the icon reverts:

  • You moved, deleted, or renamed the .ico file you used.
  • The hidden desktop.ini file that stores the setting was deleted or edited.
  • Cloud sync (like OneDrive) overwrote desktop.ini.

The fix is to keep the reference intact: put the .ico somewhere it won't move (ideally inside that folder), and don't delete desktop.ini. Synced folders are especially prone to this, so consider excluding them from sync — or using a method that doesn't rely on an external file.

The root cause is that it only "remembers a location."
A method that keeps the image internally won't revert, even if you move or delete the original.

SYMPTOM 3You can't select a photo (JPG / PNG)

You open "Change Icon," but your photos don't appear in the list. That's by design. The built-in tool only accepts the .ico format (plus some .exe / .dll files). JPGs and PNGs from your phone or camera can't be selected directly.

You have two options:

  • Convert the image to .ico using a converter tool (an extra step every time).
  • Use an app that doesn't need conversion and just drag the photo in.

SYMPTOM 4"Change Icon" is missing or greyed out

If Properties has no "Customize" tab or no "Change Icon" button, the folder may not be eligible:

  • You're on a special folder (Documents, Pictures) or a Library — the setting lives elsewhere.
  • It's a network drive, read-only, or you lack permissions — the change can't be written.

Right-click a regular folder → Properties → the Customize tab, and "Change Icon" will be there. If a folder has no Customize tab, the built-in method can't change it.
